SPEED IS OF THE ESSENCE.

This site is the ultimate embodiment of that spirit, the eternal drive to make beautiful websites that perform no matter what. How do I do it?

FILES? ONE.

A single file, providing every bit of HTML, CSS, and SVG file that this page needs. You've gotta admit, it looks really nice.

On a more serious note, having everything put into a single file reduces HTTP requests to one, meaning that your computer only sends one request and only gets one response.

A SINGLE FILE, MINIFIED*

Minification makes things fast by making things small. By taking this page's files and stripping out as much unneeded text as possible, the files become so small that transmission is nearly instant. Minified code is less readable, but is that really a problem for a computer?

SVG IS THE WAY

SVGs are compact little files. Using them allows me to save on file size while still providing nice, appealing visuals. Using them as a data URI lets me keep even those inline as well, maintaining this page's one-file-ness.

THE CONTENT DELIVERY ACCELERATOR

This webpage is hosted on a GitHub Pages server and cached with the Cloudflare CDN, a globe-spanning content delivery network that accelerates websites with servers around the world. That close physical proximity allows this page to be delivered faster to any end user.
